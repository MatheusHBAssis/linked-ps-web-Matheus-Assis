# linked-ps-web-Matheus-Assis
Landing page para processo seletivo Linked

Nome: Matheus Henrique Barbosa Assis

Uma landing page simples e responsiva para um consultório de psicologia, com seções claras — Início, Serviços, Horários, Feedbacks e Contato — imagens decorativas, mapa e formulário de contato. O objetivo é apresentar os serviços, facilitar agendamentos e oferecer navegação rápida por âncoras, com atenção à experiência em desktop e mobile.

Como abrir o projeto localmente

Método 1 — 
Abrindo diretamente no navegador
Baixe ou clone o projeto.
Navegue até a pasta do projeto.
Clique duas vezes no arquivo index.html.
O navegador abrirá a página automaticamente.

Método 2 — 
Utilizando o Visual Studio Code
Abra a pasta do projeto no Visual Studio Code.
Instale a extensão Live Server.
Clique com o botão direito em index.html.
Selecione Open with Live Server.
O site será aberto automaticamente no navegador padrão.

Tecnologias utilizadas:

Linguagens de Desenvolvimento -
HTML5: utilizado para estruturar o conteúdo da página, incluindo seções, menus, formulários e elementos de navegação.
CSS3: utilizado para estilização da interface, definição de layouts, cores, tipografia, responsividade e efeitos visuais.
JavaScript: utilizado para adicionar funcionalidades e comportamentos dinâmicos à página.
Bibliotecas e Recursos Externos
Google Fonts: utilizada para importar a fonte Inria Serif, responsável pela identidade visual do projeto.
Ferramentas de Desenvolvimento -
Visual Studio Code (VS Code): ambiente de desenvolvimento utilizado para criação e edição dos arquivos do projeto.
Live Server (extensão do VS Code): utilizado para executar o projeto localmente durante o desenvolvimento.
Git: para controle de versão e gerenciamento das alterações no código.
